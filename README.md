# Tab Close

A Chrome new tab dashboard that helps you close tab clutter and manage quick todos.

## Credits

Original project: Tab Out by Zara  
https://github.com/zarazhangrui/tab-out

This version (Tab Close) is modified and extended by Yobot.

## Features

- View all open tabs by domain
- Detect duplicate tabs
- Close tabs or groups
- Save tabs for later
- Right sidebar Todo List
- 100% local, no server
